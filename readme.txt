sitecheck.py

[概要]
・指定されたサイトをチェックして3回連続エラーコードを返してきた場合に指定されたメールアドレスにアラートメールを飛ばします。
・チェックするサイトは複数指定することが可能です。
・アラートメールを飛ばす先も複数指定することが可能です。

[動作環境]
・pythonがインストールされているマシンで動作します。

[使用方法]
1.定義部分を修正して実行してください。
1.1.47行目s = smtplib.SMTP('127.0.0.1')
　SMTPサーバのアドレスを指定してください。
1.2.60行目from_addr = 'sitecheck@****.co.jp'
　アラートメールのFromを指定してください。
1.3.61行目to_addrs = ('xxx1@****.co.jp', 'xxx2@****.co.jp',)
　アラートメールのToを指定してください。
1.4.63行目map(check, ('http://www.****.co.jp', 'http://www.****.com/****.jsp', 'http://www.******.jp/****.html'))
　チェックをするURLを指定してください。
